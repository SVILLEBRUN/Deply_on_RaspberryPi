My Project Portfolio

Here I present the different projects that have allowed me to learn and experiment with technologies such as Vue.js, Node.js, Ruby on Rails, MongoDB, PostgreSQL...

Movie DB

Vue.js | Quasar

A simple movie database application built to experiment with Vue.js and Node.js. It allows browsing and managing a list of movies with a lightweight and responsive front-end.

View Project GitHub

Movie DB API

Node.js | Express | MongoDB

A flexible backend API for managing movie data, developed with Node.js, Express, and MongoDB. Features built-in caching and indexing for optimized performance and scalability, along with a secure authentication system that can be integrated with both frontend and backend applications.

View API GitHub

AirJewels

Ruby on Rails | PostgreSQL

An Airbnb-style web platform for renting and booking jewelry. Developed by a team of 4 in 4 days during the Le Wagon bootcamp. Features messaging, booking, item listing, photo uploads, and geolocation.

View Project GitHub

Academint

Ruby on Rails | PostgreSQL

A platform to issue and verify blockchain-based certificates for students. Built in 2 weeks during the Le Wagon bootcamp using Ruby on Rails and Web3 APIs, with a focus on secure issuance and public verification.
